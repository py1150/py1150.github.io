Developer Survey Insights

May 19, 2019
Image

This blog post comprises analyses based on the stack overflow survey data 2017. In particular, two areas of investigation are focused on: A) Developer Types and B) factors for career satisfaction of developers.

A: Developer Types

In this section, it is explored whether developer types use different technologies and which salary developmentss are typical for each developer type.

Q1: Which technologies are used by which developer type?

In order to find out which programming languages are typically used by different developer types, two categories, i.e., Data Scientist and Web Developer are analyzed. The table below contains the results and indicates how many respondents are using the listed technologies.

Python R SQL Java Javascript
Data_Scientist 0.3858 0.1833 0.4462 0.5547 0.4495
Web_Developer 0.1788 0.0984 0.4594 0.6881 0.6285

From the results, it can be seen that the programming languages "Python" and "R" are typical instruments used by Data Scientists. It is used by 38.6% and 18.3% of Data Scientists respectively whereas corresponding values for Web Developers are half or less. The usage of SQL is equally distributed among the two groups. While around half of Data Scientists are familar with "Java" and "Javascript", these languages are clearly the domain of Web Developers as 60% or above of Web Developer respondents are familiar with these technologies.
