Project motivation
-------------------
Programmer/Developer is a very distinct career path. However, within this career path, many different types of developers exist.
The stack overflow survey data was used to investigate in which aspects differences between these types can be found. 


Results Summary
----------------
A Developer Types
- AQ1: Developer types differ by used technologies. Most popular: Web Developers: Javascript; Data Scientists: Python
- AQ2: Developer Salaries:
	Average salaries are very similar for the investigated developer types.
	Salaries are found to clearly increase with progressing experience for all developer types.
	Differences between developer types can be found in terms of Minimum/Top salaries as well as expected increase over experience years.
B Career Satisfaction 
- Salary, Stack Overflow Satisfaction, working hours, experience and job security are found to be the main drivers for career satisfaction
